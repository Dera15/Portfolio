Data Cleaning Projects Portfolio
Overview

This repository showcases two Microsoft Excel data-cleaning projects focused on transforming raw, inconsistent datasets into structured, analysis-ready formats. These projects demonstrate essential data preparation skills used in real-world analytics workflows.


Project 1: U.S. Presidents Data Cleaning (Project On Data Cleaning)

Objective

Cleaned and standardized a historical dataset containing information about U.S. Presidents to improve consistency, accuracy, and usability.

Key Tasks

- Standardized names and categorical values
- Removed formatting inconsistencies and extra spaces
- Applied consistent date and numeric formats
- Improved overall data quality and structure

Skills Demonstrated

- Data Cleaning & Standardization
- Text Manipulation
- Data Validation
- Date Formatting
- Data Quality Assessment



Project 2: Sales Data Cleaning (Data Cleaning II)

Objective

Transformed an unstructured sales dataset into a clean, organized format suitable for reporting and analysis.

Key Tasks

- Split combined records into separate columns
- Standardized data structure and formatting
- Validated numerical and percentage fields
- Organized data for easier analysis and visualization

Skills Demonstrated

- Text-to-Columns
- Data Structuring
- Data Validation
- Formatting & Standardization
- Spreadsheet Organization


Tools Used

- Microsoft Excel
- Excel Functions
- Text-to-Columns
- Find & Replace
- Data Validation
- Formatting Techniques



Key Learning Outcomes

- Cleaning and preparing raw datasets for analysis
- Identifying and resolving data quality issues
- Applying data standardization techniques
- Building analysis-ready datasets using Excel
